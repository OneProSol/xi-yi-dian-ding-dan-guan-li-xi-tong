### 一、相关技术
- 前端：`Vue`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8或jdk17`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、门店、管理员。

用户功能：1. 登录、2. 注册、3. 首页、4. 店铺信息、5. 交流区、6. 系统公告、7. 个人中心、8. 后台管理、9. 在线客服

门店功能：1. 登录、2. 个人中心、3. 店铺信息管理、4. 衣服类型管理、5. 洗衣信息管理、6. 订单信息管理、7. 订单进度管理

管理员功能：1. 登录、2. 个人中心、3. 顾客管理、4. 店家管理、5. 店铺信息管理、6. 衣服类型管理、7. 洗衣信息管理、8. 订单信息管理、9. 订单进度管理、10. 交流区、11. 系统管理
